# 🚀 Aryan Gore - Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my projects, skills, and contact information.

---

## 🌐 Live Demo

🔗 https://aryan-gore.github.io/porfolio-2/

---

## 📌 About The Project

This is my personal developer portfolio designed to present my work in a clean and professional way.  
The website includes smooth animations, responsive layout, interactive sections, and a functional contact form powered by EmailJS.

It serves as a central place where recruiters and collaborators can explore my projects and get in touch with me.

---

## ✨ Features

- 🔥 Fully Responsive Design
- 🎨 Modern UI with Smooth Animations
- 🧑‍💻 Projects Showcase Section
- 📜 About Me Section
- 📞 Contact Form with EmailJS Integration
- 🔗 Social Media Integration
